# Discord-Bot-Python
This is a Discord Python bot made by me with some basic moderation commands, and will be updated as I update my own bot. I want this to be open sourced, but if used please credit me in any way such as making a help command credit me. You will need to place your own Discord Bot Token into the placeholder at the bottom or make a secret if you are using replit, the secret name should just be "Token" or whatever it says at the bottom. You will also need to provide a Google AI Studio Key/Token for the /ask command to work, the place you put it is near the top but going down a bit. Please make an Issue request with any command that is not working or experiencing bugs, so I can fix it within a decent timeframe.
